================================================================================
                         TECHNICAL DETAILS
================================================================================

Algorithm: Linear Regression with Feature Engineering
Features Used:
  • Adsorbent type (one-hot encoded)
  • Metal type (one-hot encoded)
  • Dosage (g/L)
  • Temperature (°C)
  • pH
  • Contact time (minutes)
  • RPM
  • Initial concentration C0 (mg/L)

Target Variable: Removal Percentage (%)

Data Split: 80% training, 20% testing
Random Seed: 42 (for reproducibility)

Key Corrections Implemented:
  1. Prediction clipping: predictions = np.clip(predictions, 0, 100)
  2. Error capping: absolute_error = np.minimum(np.abs(error), 100)
  3. Metal combinations: Automated generation for all 8 specified metals
